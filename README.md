# Financial Health Assessment Tool for SMEs

This project analyzes financial data of small and medium enterprises (SMEs)
to assess profitability and financial health.

## Features
- CSV-based financial data analysis
- Revenue, expense, and profit calculation
- AI-based financial health summary
- Risk identification and recommendations
- Revenue vs Expense visualization

## Tech Stack
- Python
- Pandas
- Matplotlib
- Google Colab

## How to Run
Open the notebook in Google Colab and run all cells.
